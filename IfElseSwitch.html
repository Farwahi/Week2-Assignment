Brief on If,Else & Switch by using the validateForm

This code will display a pop-up message with the text “Yes I can serve you” if the user is older than 17 years old and the country is the United Kingdom. If the user is younger than 18 years old, the code will display a pop-up message with the text “You aren’t old enough”. If the user is older than 17 years old but not from the UK, the code will display a pop-up message with the text “Sorry, we don’t serve that country”. If the user is from the USA or Canada, the code will display a pop-up message with the text “Sorry, we don’t serve the USA” or “Sorry, we don’t serve Canada”, respectively.
